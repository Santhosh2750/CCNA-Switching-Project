# Cisco Switching Lab

## 📌 Project Overview
This project demonstrates the implementation of Layer 2 switching technologies using Cisco switches in Cisco Packet Tracer. It covers VLAN creation, trunking, Inter-VLAN communication, EtherChannel, and VTP configuration to simulate an enterprise network.

---

## 🎯 Objectives

- Configure VLANs
- Configure Access Ports
- Configure Trunk Ports
- Configure Inter-VLAN Routing
- Configure EtherChannel (Port Channel)
- Configure VTP Server and Client
- Verify connectivity between VLANs

## ⚙️ Features Implemented

### VLAN Configuration
- Created multiple VLANs
- Assigned switch ports to VLANs
- Verified VLAN membership

### Trunk Configuration
- Configured IEEE 802.1Q trunk links
- Allowed required VLANs over trunk links

### Inter-VLAN Routing
- Configured SVIs
- Assigned IP addresses to VLAN interfaces
- Enabled routing between VLANs

### EtherChannel
- Configured Port Channel using LACP
- Increased bandwidth
- Added redundancy

### VTP
- Configured VTP Server
- Configured VTP Client
- Verified VLAN synchronization

### STP Verification
- Verified Root Bridge
- Checked port states
- Prevented switching loops
